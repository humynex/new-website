DROP YOUR WORK FILES HERE
=========================

For each project, add two files:

  tld-contracting.jpg   + tld-contracting.mp4
  webflow-pro.jpg       + webflow-pro.mp4
  autoagent.jpg         + autoagent.mp4
  appforge.jpg          + appforge.mp4

JPG  = screenshot of the website (shows as the card thumbnail)
MP4  = short screen-recording / demo video (plays when the card is clicked)

If only a JPG is present with no MP4, the modal will open but no video plays.
The modal will still show the project title, description, tags, and visit link.
